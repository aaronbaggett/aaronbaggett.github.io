Setting Up a Discord Account and Joining a Server

Follow this guide to create a new Discord account and join a server using the invitation link below. Be sure to say hi to everyone with a GIF in the #general channel.

1. Creating a Discord Account

Follow these steps if you have not already created your Discord account.

  1. Open your web browser and go to Discord’s official website.
  2. Click on the Sign Up button usually located at the top right corner of the page.
  3. Enter your email address, create a username, set a password, and enter your date of birth.
  4. Click Continue to proceed.
  5. Discord will send a verification email to the address you provided.
  6. Open your email inbox, find the Discord verification email, and click on the verification link.
  7. Once your email is verified, your Discord account is ready to use.

2. Joining the PSYC 2316 Discord Server

Obtain the Discord server invitation link here: discord.gg/Zrmx3V8b and follow the on-screen instructions.

  1. If using Discord on a mobile device:
    • Open Discord.
    • Click the Add a Server button on the left sidebar (it looks like a plus + sign).
    • Choose Join a Server and paste the invitation link into the box.
    • Click Join.
  2. If using Discord in a web browser:
    • Simply click on the invitation link, and it will open a new tab in your browser with an option to Accept Invite.
    • If you are not logged in, you will be prompted to do so. After logging in, you can join the server.

3. Explore the Server

  • Once you’ve joined the server, take a look around.
  • Say hi with a GIF in the #general channel.

Tips for Using Discord

  • Customize your profile by clicking on the User Settings gear icon near your username.
  • Set up two-factor authentication for added security.
  • Explore user settings to adjust notifications, privacy settings, and more.
